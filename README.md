## 🎯 Usage Guide

### 🧭 **Navigation**
- **Desktop**: Use the top navigation bar with smooth scroll
- **Mobile**: Tap the hamburger menu for slide-out navigation
- **Search**: Use the search bar to find specific products

### 🛒 **Shopping Experience**
1. **Browse Products**: Scroll through the curated collection
2. **Product Details**: Click any product for detailed view
3. **Add to Cart**: Use the "Add to Cart" button (updates counter in real-time)
4. **Checkout**: Click cart icon and fill out the order form

### 🎨 **Interactive Elements**
- **Product Hover**: Hover over products for zoom and details
- **Image Gallery**: Navigate through product images
- **Color Variants**: Select different color options
- **Size Selection**: Choose from available sizes

## ⚙️ Customization Guide

### 🎨 **Theming**
```css
/* Edit CSS variables in style.css */
:root {
    --primary-color: #667eea;
    --secondary-color: #764ba2;
    --accent-color: #f093fb;
    --text-color: #333;
    --bg-color: #ffffff;
}
```

### 🏷️ **Products**
1. Add product images to `/assets/images/products/`
2. Update product data in `main.js`
3. Modify product HTML structure in `index.html`

### 📱 **Responsive Breakpoints**
```css
/* Mobile First Approach */
@media (min-width: 768px)  { /* Tablet */ }
@media (min-width: 1024px) { /* Desktop */ }
@media (min-width: 1440px) { /* Large Desktop */ }
```
